# Design Concepts for ClarityCoach

This folder contains three independent, production-grade visual systems for the app. Each design has a runnable Flutter showcase file so you can preview it quickly.

## 1) Quiet Focus
Tone: Calm, editorial, grounded. Built for reflective planning and weekly reviews.

Palette
- Ink: #1F1A17
- Cream: #F7F2EB
- Sage: #4B6A5C
- Terracotta: #C06B4E
- Sky: #9DB6C8

Typography
- Display: Fraunces
- Body: Source Sans 3

Motion
- Slow, subtle fades and slide-ins.
- Gentle elevation transitions for cards.

Preview
- Run: flutter run -t designs/quiet_focus.dart

## 2) Pulse
Tone: Energetic, modern, high-contrast. Built for momentum and daily execution.

Palette
- Night: #0B0D12
- Slate: #151924
- Neon Lime: #A7F72A
- Electric Blue: #4CC9FF
- Hot Magenta: #FF3E8E

Typography
- Display: Space Grotesk
- Body: Manrope

Motion
- Snappy, short duration transitions.
- Accent glows on primary actions.

Preview
- Run: flutter run -t designs/pulse.dart

## 3) Harbor
Tone: Warm, tactile, outdoors. Built for clarity and long-term stewardship.

Palette
- Deep Navy: #1C2B36
- Sand: #E9E3D6
- Teal: #2D6A6F
- Sun: #E1A65E
- Fog: #6C7A89

Typography
- Display: Cormorant Garamond
- Body: Work Sans

Motion
- Soft parallax-like section reveals.
- Rounded corners with gentle shadows.

Preview
- Run: flutter run -t designs/harbor.dart
